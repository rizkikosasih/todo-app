## 2. Deskripsi Postingan (Siap Salin)

```markdown
Bagaimana cara membuat aktivitas menyelesaikan tugas harian (To-Do List) terasa menyenangkan dan memotivasi? Jawabannya: Gamifikasi! 🎮

Saya baru saja menyelesaikan penulisan ulang total (clean rewrite) dari Todo App V1 (JavaScript & Email Login) menjadi **Todo App V2** — sebuah aplikasi pelacak tugas modern yang memadukan produktivitas dengan mekanika RPG.

Berikut adalah beberapa peningkatan fitur utama yang saya rancang di V2:
🔹 **Gamification Engine:** Setiap tugas utama memberikan +10 XP. Pengguna akan naik level setiap mencapai 100 XP, lengkap dengan letupan selebrasi konfeti visual (canvas-confetti & framer-motion).
🔹 **Daily Streak Tracker:** Logika pintar untuk melacak konsistensi harian pengguna secara berurutan. Streak akan bertambah jika pengguna aktif menyelesaikan tugas setiap harinya dan ter-reset jika terlewat satu hari.
🔹 **Subtasks & Categorization:** Kemampuan memecah tugas menjadi checklist sub-tugas independen, lengkap dengan tag kategori warna-warni dan tingkat prioritas (High, Medium, Low).
🔹 **Custom SVG Weekly Chart:** Grafik statistik produktivitas 7 hari terakhir yang digambar menggunakan SVG dinamis murni untuk menjaga ukuran bundle aplikasi tetap ringan.
🔹 **Satu Klik Google OAuth2:** Autentikasi aman terpusat menggunakan Google Sign-In Firebase Auth.
🔹 **Dark/Light Mode:** Transisi mode gelap dan terang yang halus menggunakan variabel HSL Tailwind CSS.

Dari sisi teknis, proyek ini bermigrasi penuh ke:
💻 **React TypeScript** (Strict type-safety)
state **Zustand** (State management global yang super ringan)
🎨 **Shadcn UI & Tailwind CSS** (Desain antarmuka premium & responsif)
🔥 **Firebase Web SDK v10** (Authentication & Realtime Database)

Proses pengembangan ini melatih saya dalam merancang arsitektur aplikasi yang bersih, memisahkan logika database/state dari tampilan UI, serta menangani optimasi build produksi.

Silakan cek repositori kode lengkapnya di sini: [Masukkan Link Github Anda]
```

## 3. Keahlian & Keterampilan Utama (Skills)

Berikut adalah daftar keahlian (_Skills_) yang dapat Anda tambahkan langsung di LinkedIn untuk proyek ini:

### Keahlian Teknis (Hard Skills):

1. **React.js** – Untuk merancang antarmuka modular dan interaktif.
2. **TypeScript** – Untuk meningkatkan keamanan tipe data (_type safety_) dan mencegah bug runtime.
3. **Zustand** – Untuk manajemen state global yang efisien dan ringan.
4. **Tailwind CSS** – Untuk pembuatan sistem gaya (_styling_) responsif dan transisi HSL.
5. **Firebase Web SDK** – Integrasi Google OAuth2 dan sinkronisasi real-time database.
6. **Git & GitHub** – Untuk manajemen branch (`feat/`), tag rilis (`v1.0.0`), dan version control terstruktur.
7. **PostCSS** – Pemrosesan kompilasi CSS modern.

### Keahlian Metodologi & Desain (Soft/Process Skills):

1. **UI/UX Design** – Merancang antarmuka bertema gelap/terang premium dengan efek glassmorphic.
2. **Gamification Design** – Merancang mekanika permainan (XP, Level, Streak) untuk meningkatkan retensi pengguna.
3. **Refactoring & Code Migration** – Pengalaman memigrasikan codebase JavaScript (legacy) ke arsitektur modern TypeScript.
